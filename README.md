# aoj-submit.el

An [AOJ](http://judge.u-aizu.ac.jp/onlinejudge/) client for Emacs.

## Installation

```elisp
(require 'aoj-submit)
(setq aoj-id "Your ID")
(setq aoj-password "Your Password")
```

## How to use

Make sure that the filename is "problemNO(or LessonID).cpp".

`M-x aoj-submit` to submit.
